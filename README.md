# lendsqr-fe-test

## Overview

This project is a frontend implementation of the Lendsqr admin dashboard using React, TypeScript, and SCSS.

It includes:
- Login page
- Dashboard page
- Users page
- User details page

The users page pulls data from a mock API source with 500 records, and selected user details are stored and retrieved with localStorage.

## Features

- React + TypeScript application
- SCSS styling
- Routing with React Router
- Mock API using `public/users.json`
- 500 generated user records
- User filtering by status
- Row action menu
- User details page
- localStorage persistence
- Responsive layout

## Tech Stack

- React
- TypeScript
- Vite
- React Router DOM
- SCSS
